# Промпт: Разработка сайта для продажи сантехники

Ты — опытный фронтенд-разработчик. Тебе нужно создать полноценный, адаптивный, одностраничный сайт-каталог для частного предпринимателя, который продает сантехнику.

## Общие требования

1.  **Стек технологий:** Чистый HTML, CSS, JavaScript. Без фреймворков и библиотек (кроме FontAwesome для иконок).
2.  **Структура:** Всё в одном файле `index.html` (стили в `<style>`, скрипты в `<script>` в конце body).
3.  **Адаптивность:** Сайт должен корректно отображаться на всех устройствах (десктоп, планшет, смартфон). Использовать CSS Grid и Flexbox.
4.  **Хранение данных:** LocalStorage для корзины и списка избранных товаров.

## Структура и контент (секции)

### 1. Шапка сайта (Header)
- **Логотип:** Текст "СантехникаПро" с иконкой крана (FontAwesome).
- **Навигация:** Меню из пунктов: "Каталог", "О нас", "Контакты". (На мобильных устройствах меню сворачивается в гамбургер).
- **Контакты:** Номер телефона `+7 (999) 123-45-67` с иконкой.
- **Корзина:** Иконка корзины с счетчиком (количество товаров). При клике открывается выпадающее окно с содержимым корзины.

### 2. Главный экран (Hero Section)
- Фоновый градиент (от темно-синего к синему).
- Заголовок: "Качественная сантехника для дома и бизнеса".
- Подзаголовок: "Смесители, ванны, душевые кабины и аксессуары с гарантией качества".
- Кнопка: "Перейти в каталог" (якорь на секцию каталога).

### 3. Каталог товаров (самая важная часть)

**Данные о товарах:**
Создай массив `products` из 10 товаров со следующими полями:
- `id` (число)
- `name` (строка)
- `category` (строка: "смесители", "ванны", "душевые", "унитазы", "аксессуары")
- `price` (число)
- `oldPrice` (число, опционально, если есть скидка)
- `image` (строка, ссылка на картинку-заглушку через `https://via.placeholder.com/300x200?text=Название`)
- `rating` (число от 1 до 5)
- `isHit` (boolean, true если товар популярный)
- `isNew` (boolean, true если новинка)

**Функционал каталога:**
1. **Фильтрация по категориям:**
   - Вывести кнопки-фильтры: "Все", "Смесители", "Ванны", "Душевые кабины", "Унитазы", "Аксессуары".
   - При клике на кнопку показываются товары только выбранной категории. Активная кнопка должна визуально выделяться (другой цвет или подчеркивание).

2. **Сортировка:**
   - Выпадающий список (`<select>`) с опциями: "По умолчанию", "Сначала дешевле", "Сначала дороже", "По популярности (рейтингу)".

3. **Отображение:**
   - Карточки товаров в сетке (4 колонки на десктопе, 2 на планшете, 1 на телефоне).
   - В каждой карточке: изображение, название, цена (если есть старая цена, перечеркнуть её), рейтинг (звездочками), бейджи "🔥 Хит" или "Новинка" (если соответствующий флаг true).
   - Кнопка "В корзину" и кнопка-сердечко "В избранное".

### 4. Избранное
- При клике на сердечко оно меняет цвет (становится красным/заполненным).
- Состояние избранного сохраняется в LocalStorage, чтобы после перезагрузки страницы сердечки оставались отмеченными.

### 5. Корзина
- При клике на кнопку "В корзину" товар добавляется в массив корзины.
- Счетчик на иконке корзины обновляется.
- При клике на иконку корзины открывается выпадающее окно (dropdown) со списком товаров в корзине. У каждого товара есть кнопка "Удалить" (крестик). Внизу показана общая сумма.
- Данные корзины также сохраняются в LocalStorage.

### 6. Секция "О нас" (преимущества)
- 4 блока с иконками (FontAwesome) и текстом:
  - 🔧 Оригинальная продукция
  - 🛡 Гарантия до 5 лет
  - 🚚 Бесплатная доставка от 10 000 ₽
  - 💬 Помощь в подборе

### 7. Секция "Контакты и форма обратной связи"
- Телефон, email, адрес (можно вымышленный, например, г. Москва, ул. Строителей, 15).
- Форма с полями:
  - Имя (текстовое поле)
  - Телефон (поле с маской: +7 (___) ___-__-__)
  - Комментарий (textarea)
- Кнопка "Отправить".
- **Валидация:** Проверить, что поле телефона заполнено правильно. Если ошибка — подсветить красным и показать сообщение.
- При успешной отправке вывести данные в консоль и показать alert "Спасибо! Мы свяжемся с вами".

### 8. Подвал (Footer)
- Копирайт "© 2025 СантехникаПро".
- Ссылки на соцсети (VK, Telegram, WhatsApp) — просто иконки без ссылок.

## Дизайн и стилизация
- **Цвета:**
  - Основной фон: `#f8fafc` (светлый)
  - Шапка и подвал: `#1e293b` (темно-синий)
  - Акцентный цвет для кнопок и хитов: `#f59e0b` (золотистый/оранжевый)
  - Карточки: белый фон, тень `0 4px 6px rgba(0,0,0,0.1)`
- **Шрифт:** Подключить Google Font "Roboto" или "Inter".
- **Эффекты:**
  - При наведении на карточку: небольшое увеличение (`transform: scale(1.02)`) и изменение тени.
  - Плавная анимация появления карточек при загрузке (fadeIn).
- **Прелоадер:** Спиннер, который показывается 1 секунду при загрузке страницы, затем исчезает.

## Интерактивность (JavaScript)
- Все фильтры, сортировки, добавление в корзину и избранное работают **без перезагрузки страницы**.
- Используйте делегирование событий для карточек товаров (события на родителе).
- При клике на "В корзину" или сердечко должен быть визуальный фидбек (например, кнопка меняет цвет на секунду).

## Техническое задание для OpenCode
- Создай один файл `index.html`.
- Внедри весь CSS в тег `<style>` внутри `<head>`.
- Весь JavaScript размести в конце `<body>` в теге `<script>`.
- Код должен быть чистым, с комментариями на русском языке (поясняющими основные блоки).

## Дополнительно (опционально, но будет плюсом)
- Добавь на главный экран **таймер обратного отсчета** до окончания акции (например, до конца недели).
- Сделай плавный скролл по якорям с помощью CSS (`scroll-behavior: smooth`).
